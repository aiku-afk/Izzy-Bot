# IzzyBot WhatsApp

Bot de WhatsApp usando Baileys.

## Instalação no Termux

1. Atualizar Termux

pkg update && pkg upgrade

2. Instalar dependências

pkg install nodejs ffmpeg yt-dlp git

3. Instalar dependências do bot

npm install

4. Rodar bot

node bot.js

## Rodar com PM2

npm install -g pm2

pm2 start bot.js
